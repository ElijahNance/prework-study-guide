HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Elements inside other elements are called child elements, the element that holds these elements is the parent element
  • Metadata is data about the webpage that the browser uses to effectively serve the webpage to our users
  • We use Semantic HTML elements to indicate the role of their content, as well as to give structure to the content of the webpage

CSS

  • CSS is a programming language used to define how webpage content is presented to users
  • Inline CSS is used to style an element directly into the file
  • Internal style sheets are used to embed an entire stylesheet directly into the HTML file
  • External CSS style sheets allow developers to keep all their CSS rules in a seperate file
  • Seperation of code refers to seperating code into different files based on use
  • A selector defines the element or atrtributes to which the rules, or declarations will apply
  • Declaration contain two components: the CSS property we want to apply and the value of the property
  • Margin properties indicate how much space we want around the outside of an element
  • Padding adds space around the content inside an element
  • Do Not Repeat Yourself(DRY) is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the nummber of lines of code written
  • Class attributes allow you to share a CSS rule to any element you choose by assigning the rule to a class attribute with a class selector

Git

  • Git is a version control system that lets you manage and keep track of each version of your project
  • Git commands are used to save and track versions of your code
  • GitHub is a cloud-based hosting service that allows you to keep track of your projects inside repositories that are stored remotely
  • GitHub issues give devolopers and teams an easy way to keep track of key tasks and progress
  • A branch is a copy or version of the code that can be worked on independently in isolation so development changes do not disrupt other team members who are also working on the code

JavaScript

  • JavaScript is a high-level dynamic programming language that allows users to interact with websites
  • Variables are containers that store values, you start by declaring a variable with the "let" keyword
  • Strings are a sequence of text signified with single quotation marks
  • Boolean is a true/false statement
  • Arrays allow you to store multiple values in a single reference
  • Comments are snippets of text that can be added along with code
  • An operator is a mathematical symbol that produces a result based on two values
  • Conditionals are code structures used to test if an expression returns true or not
  • Functions are a way of packaging functionality that you wish to reuse
  • Control flow is the order in which a computer executes code in a script
  • You can have more then one condition by using else-if
  • For loops use the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over
