HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Elements inside other elements are called child elements, the element that holds these elements is the parent element
  • Metadata is data about the webpage that the browser uses to effectively serve the webpage to our users
  • We use Semantic HTML elements to indicate the role of their content, as well as to give structure to the content of the webpage

CSS

  • CSS is a programming language used to define how webpage content is presented to users
  • Inline CSS is used to style an element directly into the file
  • Internal style sheets are used to embed an entire stylesheet directly into the HTML file
  • External CSS style sheets allow developers to keep all their CSS rules in a seperate file
  • Seperation of code refers to seperating code into different files based on use
  • A selector defines the element or atrtributes to which the rules, or declarations will apply
  • Declaration contain two components: the CSS property we want to apply and the value of the property
  • Margin properties indicate how much space we want around the outside of an element
  • Padding adds space around the content inside an element
  • Do Not Repeat Yourself(DRY) is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the nummber of lines of code written
  • Class attributes allow you to share a CSS rule to any element you choose by assigning the rule to a class attribute with a class selector
